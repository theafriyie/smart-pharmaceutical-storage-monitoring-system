# Hardware

This folder contains information related to the physical hardware used in the smart pharmaceutical storage monitoring and control system.

## Hardware Components

- Arduino Uno R3
- ESP8266 NodeMCU v1.0
- DHT22 temperature and humidity sensor
- 1-channel 5V relay module
- 16x2 I2C LCD display
- DC cooling fan
- 5V regulated power supply
- Jumper wires
- Prototype enclosure

## Hardware Role

The hardware layer is responsible for environmental sensing, decision execution, local display, and cooling actuation within the pharmaceutical storage prototype.
