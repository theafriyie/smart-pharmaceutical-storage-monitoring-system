# Mobile App

This folder contains materials related to the Flutter-based mobile application used in the smart pharmaceutical storage monitoring and control system.

The app was designed to help facility personnel monitor storage conditions remotely and respond quickly to unsafe environmental changes.

## App Features

- User authentication
- Real-time temperature display
- Real-time humidity display
- Alert notification log
- Historical data and trend viewing
- Remote monitoring access

## Technologies

- Flutter
- Firebase Authentication
- Firebase Realtime Database
- Firebase Cloud Messaging
